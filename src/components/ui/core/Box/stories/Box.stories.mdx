# Box

#### Box component serves as a component for using <code>div</code> <code>span</code> <code>aside</code> <code>main</code> <code>header</code> <code>section</code> <code>footer</code>. Just pass these elements inside as prop which will render the element. It will accept all the props accepted by the element.
