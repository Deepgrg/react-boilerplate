## Variants

#### Text component uses variant mapping, to map guideline styles to the variant. If variant is not passed defaults to paragraph. Available variants are,

### <code>h1</code> <code>h2</code> <code>h3</code> <code>h4</code> <code>h5</code> <code>h6</code> <code>paragraph</code> <code>label</code> <code>small</code> <code>overline</code> <code>subtitle1</code> <code>subtitle2</code> <code>subtitle3</code>

<Code>
  {`<Text variant={'h1'}>This is h1 Variant</Text>`}
</Code>

<Canvas>
  <Story name="Variants">
    <Text variant={'h1'} className={'my-5'}>This is h1 Variant</Text>
  </Story>
</Canvas>

## AS

#### Text component default variant element mapping can be changed by passing element to the <code>as</code> prop. If not massed default mapping is used. Lets say you have h1 tag already in the page, but want h1 styles without another h1 element, just pass the tag you want to render to the <code>as</code> prop.

### <code>h1</code> <code>h2</code> <code>h3</code> <code>h4</code> <code>h5</code> <code>h6</code> <code>p</code> <code>label</code> <code>small</code> <code>overline</code> <code>span</code>

<Code>
  {`<Text variant={'h1'} as={'h2'}>This is h1 Variant, rendered as h2 element</Text>`}
</Code>

<Canvas>
  <Story name="AS">
    <Text variant={'h1'} className={'my-5'}>This is h1 Variant, rendered as h2 element</Text>
  </Story>
</Canvas>

## Align

#### Text Align Prop defines the <code>text-align</code> css property.
### <code>center</code> <code>inherit</code> <code>justify</code> <code>left</code> <code>right</code> <code>start</code> <code>end</code>

<Canvas>
  <Story name="Align">
    <Text variant={'paragraph'} align={"right"} className={'my-5'}>This is paragraph Variant, aligned to right</Text>
  </Story>
</Canvas>

## Typeface

#### Typeface prop defines the prebuilt font weight of the text.
### <code>thin</code> <code>extralight</code> <code>light</code> <code>normal</code> <code>medium</code> <code>semibold</code> <code>bold</code> <code>extrabold</code> <code>black</code>
